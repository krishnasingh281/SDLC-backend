# Intelligent SDLC Assistant

AI-powered backend with 9 specialist APIs for software teams: design trade-offs, design reviews, risk analysis, tests, compliance, and more. Built with **Flask** + **Gemini** (Phase 1) and a custom NLP model (Phase 2).

## Features
- Design Trade-Off Advisor (Monolith vs Microservices, etc.)
- Design Review Assistant (summary, risks, action items)
- Risk Analysis Engine (JSON table with scoring)
